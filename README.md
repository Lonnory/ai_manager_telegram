# 🤖 AI Manager Bot (@ai_massage_manager_bot)

[![Telegram Bot](https://img.shields.io/badge/Telegram-Bot-blue.svg?logo=telegram)](https://t.me/ai_massage_manager_bot)

Мощный Telegram-бот для управления бизнес-аккаунтами (Telegram Business), который берет на себя рутину общения с клиентами. Бот работает как продвинутый автоответчик и ИИ-ассистент, полностью интегрированный с API Nvidia, поддерживая балансировку нагрузки, шифрование данных и работу с медиа.

---

## 🌟 Что умеет бот?

*   **🤖 Полноценный ИИ-режим (`.gpt`)**: Бот может вести непрерывный диалог с пользователем от лица менеджера, запоминая контекст. Поддерживает анализ изображений!
*   **❓ Разовые ответы (`.ask`)**: Возможность задать ИИ быстрый вопрос и переслать ответ пользователю без включения постоянного контекста.
*   **⚖️ Load Balancing & Failover**: Встроенный пул из нескольких API-ключей Nvidia. Если один ключ исчерпывает лимит или выдает ошибку, бот мгновенно и незаметно переключается на следующий (Least Connections).
*   **🛡️ Полная безопасность и шифрование**: Все переписки, имена пользователей и медиа надежно шифруются локально с помощью алгоритма AES-GCM.
*   **🔕 Управление мутом (`.mute` / `.mute off`)**: Временная или полная блокировка назойливых пользователей.
*   **🛎️ Автоответчик для менеджера**: Если ИИ-режим выключен, бот сам оповестит клиента, что ему ответят позже (раз в 30 минут), и отправит уведомление владельцу аккаунта в ЛС.

## 🛠️ Как это работает?

1.  Бот подключается к вашему аккаунту через **Telegram Business Connection**.
2.  Перехватывает входящие сообщения от собеседников.
3.  В зависимости от настроек чата: либо отвечает через ИИ (с потоковой генерацией ответа), либо включает автоответчик "ожидания менеджера".
4.  Вся история сообщений надежно шифруется и складывается в локальную базу данных `messages.db` на сервере.
5.  Бот устойчив к перебоям сети: при кратковременном отключении интернета он **автоматически переподключается** и возобновляет работу без вашего вмешательства.

## 🔗 Как подключить бота к своему аккаунту?

Вы можете использовать уже готового бота **[@ai_massage_manager_bot](https://t.me/ai_massage_manager_bot)**:
1. Перейдите в **Настройки** своего профиля Telegram.
2. Нажмите **Изменить** (или иконку карандаша).
3. Найдите раздел **Автоматизация чатов** (Telegram Business).
4. Нажмите **Добавить бота** и укажите юзернейм: `@ai_massage_manager_bot`.
5. Бот моментально подключится и начнет обрабатывать ваши чаты!

## 🚀 Как запустить бота?

### 1️⃣ Подготовка (Общая для всех ОС)
1. Установите **Python 3.10+**.
2. Создайте своего бота через **[@BotFather](https://t.me/BotFather)** и получите токен. 
   *⚠️ ВАЖНО: Обязательно перейдите в настройки бота в BotFather (Bot Settings) и включите **Business Mode** (Turn on Business Mode), иначе бот не сможет подключаться к бизнес-аккаунтам!*
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Создайте файл `.env` в корневой папке проекта со следующим содержимым:
   ```env
   BOT_TOKEN=ваш_токен_бота_от_BotFather
   AI_API_POOL=ключ1,ключ2,ключ3
   INVOKE_URL=https://integrate.api.nvidia.com/v1/chat/completions
   AI_MODEL=minimaxai/minimax-m3
   ```

### 🐧 Запуск на сервере (Linux)

Проект полностью адаптирован для работы на Linux: все пути к базам данных и настройкам рассчитываются абсолютно. 

Рекомендуется запускать через **systemd**, чтобы бот работал в фоновом режиме и автоматически перезапускался.

1. Создайте файл службы:
   ```bash
   sudo nano /etc/systemd/system/ai_manager.service
   ```
2. Вставьте туда конфигурацию:
   ```ini
   [Unit]
   Description=AI Message Manager Bot
   After=network.target

   [Service]
   User=root
   WorkingDirectory=/путь/к/папке/бота
   ExecStart=/usr/bin/python3 /путь/к/папке/бота/main.py
   Restart=always

   [Install]
   WantedBy=multi-user.target
   ```
3. Запустите бота:
   ```bash
   sudo systemctl daemon-reload
   sudo systemctl enable ai_manager
   sudo systemctl start ai_manager
   ```

### 🪟 Запуск на ПК (Windows)

Для локального запуска и тестирования бота на Windows:

1. Откройте командную строку (`cmd`) или PowerShell.
2. Перейдите в папку с ботом:
   ```cmd
   cd C:\путь\к\папке\ai_profile_manager
   ```
3. Запустите главный скрипт:
   ```cmd
   python main.py
   ```
4. Бот начнет работу (вы увидите логи запуска в консоли). Чтобы остановить его, просто нажмите `Ctrl+C`.

## 🎮 Доступные команды (только для владельца)

Отправляйте эти команды в чат с клиентом (они будут удалены ботом, а клиент их не увидит):

*   `.gpt` — Включить ИИ-режим (или выключить, если он уже включен).
*   `.ask [вопрос]` — Задать разовый вопрос нейросети и отправить ответ в чат.
*   `.mute [минуты]` — Замутить собеседника (или снять мут при повторном вводе).
*   `.help` — Вывести красивое меню со списком команд.
